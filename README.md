Veamos los resultados y pensemos ¿Qué significa className en React? ¿las props tienen un limite? ¿Quién define las
props?

Respuesta 
¿Qué significa className en React?
className es el atributo que se usa en React para asignar clases CSS a los elementos, ya que la palabra class es reservada en JavaScript.

¿Las props tienen un límite?
No hay un límite fijo para la cantidad de props que puedes pasar a un componente, pero es recomendable mantenerlas organizadas para facilitar el mantenimiento del código.

¿Quién define las props?
Las props son definidas por el componente padre cuando utiliza (invoca) al componente hijo, pasando los valores como atributos en JSX.
